# Contributing to DevPulse

Cảm ơn bạn đã quan tâm đến việc đóng góp cho **DevPulse**.

Chúng tôi hoan nghênh mọi hình thức đóng góp:

* Bug fixes
* Feature improvements
* Documentation updates
* Performance optimizations
* New tools/modules

---

# Development Setup

Clone repository:

```bash
git clone https://github.com/manucian-official/devpulse.git
```

Di chuyển vào project:

```bash
cd devpulse
```

---

# Development Workflow

Tạo branch mới:

```bash
git checkout -b feature-new-feature
```

Sau đó commit:

```bash
git add .
git commit -m "feat: add new feature"
```

Push lên GitHub:

```bash
git push origin feature-new-feature
```

Tạo **Pull Request** để merge vào branch chính.

---

# Commit Convention

DevPulse sử dụng **Conventional Commits**:

```
feat: new feature
fix: bug fix
docs: documentation changes
refactor: code refactoring
chore: maintenance tasks
```

Ví dụ:

```
feat(scanner): improve project file scanning
```

---

# Code Style

* Tuân theo **PEP8**
* Code phải rõ ràng và dễ đọc
* Comment cho logic phức tạp

---

# Reporting Issues

Nếu bạn gặp lỗi, hãy tạo **Issue** với:

* Mô tả vấn đề
* Các bước để tái hiện lỗi
* Environment (Python version, OS)
